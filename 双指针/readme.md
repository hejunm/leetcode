# 双指针算法

## 适用问题类型

* 从有序序列中查找符合条件的数据集合

## 解题模式

* 必须是有序序列， 如果无序 需先排序
* 初始化left，right指针 （例如：指向头尾）
* 根据条件，移动left 或者 right （同向、相向）
